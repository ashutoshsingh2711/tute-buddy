## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/tutebuddy.git
```

2. Navigate to the project
```bash
cd tutebuddy
```

3. Open `project.html` in your web browser

## Deployment

This project can be deployed on:
- **GitHub Pages** - Free hosting from your GitHub repository
- **Netlify** - Easy continuous deployment
- **Vercel** - Fast and reliable hosting
- **Traditional Web Server** - Any standard web hosting

### GitHub Pages Deployment

1. Push your code to GitHub
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/tutebuddy`

### Netlify Deployment

1. Connect your GitHub repository to Netlify
2. Netlify will automatically deploy on every push

### Vercel Deployment

1. Import your GitHub repository to Vercel
2. Configure build settings (no build needed for static HTML)
3. Deploy
